# AImplement Training Portal

Premium static training portal for GitHub Pages with shared Firebase Realtime Database data.

## Upload to GitHub

Upload every file in this folder to the root of the `AImplement-Training` repository:

- `index.html`
- `admin.html`
- `dashboard.js`
- `admin.js`
- `styles.css`
- `class-logo.png`

Public dashboard:
`https://ab11206.github.io/AImplement-Training/`

Admin page:
`https://ab11206.github.io/AImplement-Training/admin.html`

Admin PIN: `5339`

## Included features

- Shared live Firebase data
- 8 class session cards with animated charts
- Attendance and review lists
- Overall statistics and follow-up counts
- Student directory and profile history
- Leaderboard
- Attendance and review analytics
- Session status, dates, instructor notes, and resource links
- One-click mark all present / reviewed
- JSON export and import backup
- Light and dark themes
- Responsive mobile, tablet, and desktop design
- Automatic last-updated timestamp and updater name

## Firebase rules

Firebase test-mode rules expire. Keep read/write enabled for this simple PIN-based setup or replace them with more restrictive rules later.


Update v3: Admin bulk attendance/review actions now apply only to the class selected in the Class dropdown.
